# GROWTH.md - Codeg 增长策略

> 企业级多 Agent 编码工作台的增长路线图

## 📍 定位策略

### 核心定位
**"AI 编码的统一入口"** — 一个平台管理所有 AI 编码 Agent

### 差异化优势
| 维度 | Codeg | 竞品 |
|------|-------|------|
| Agent 数量 | 5+ (Claude Code, Codex, OpenClaw...) | 通常 1-2 |
| 远程开发 | ✅ 浏览器即可使用 | 大多仅本地 |
| 会话管理 | 统一聚合 | 分散 |
| Git 工作流 | 内置 worktree | 需外部工具 |

### 目标用户
1. **企业开发团队** — 需要统一管理 AI 工具
2. **远程开发者** — 需要浏览器访问本地环境
3. **AI 编码重度用户** — 同时使用多个 Agent
