核心路径

  1. 恢复最近会话与事件游标
  2. 查看执行步骤或待处理入口
  3. 在会话内批准、拒绝或回答
  4. 继续执行并回填结果

开发交付

  • 长会话使用虚拟列表，上滑分页不改变阅读锚点。
  • WebSocket 重连携带 sessionId 与 lastEventId，避免重复工具调用。
  • Token 存入 Keychain 或 Keystore，不进入日志与通知。
  • 输入区消费安全区和软键盘 Insets，正文保持 16px。
  • 中英文与动态字体均允许容器增高，不截断主操作。
